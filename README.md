## Introduction

This is the remastered version of "microemu", a Java Swing based desktop J2ME emulator.

## Usage

```
cd microemulator 
mvn package -Dmaven.test.skip
java -jar microemulator/target/microemulator-3.0.0-SNAPSHOT.jar
```
